# 队员个人页（Player Dashboard）改版设计文档

## 背景与目标

现有队员个人页（`PlayerDashboardPage.vue`）采用暗黑奢华风格（深色背景 `#0D0D0D` + 金色点缀），与大屏页面的明亮卡通风格不一致。本次改版目标：

1. **视觉风格统一**：与大屏页面保持一致（明亮卡通、天空蓝到草地绿渐变、白色圆角卡片）
2. **功能整合**：将 FIFA 风格球员卡（六维雷达图、综合评分、维度分数）融入个人页，使个人页成为完整的"学员档案页"
3. **多场景适配**：同时满足教室智慧大屏展示、家长手机查看、教练手机查看三种场景

## 访问流程与业务规则

### 访问来源
- **大屏点击**：学员在大屏上点击自己的宠物卡 → 进入个人页
- **直接访问**：家长/教练通过链接直接访问 `/player/:playerId`

### 模式控制规则（重要）
- 大屏页面（`/screen?c=教练手机号`）需要获取教练的 `playerMode`
- **开放模式（`open`）**：大屏宠物卡可点击，点击后进入个人页
- **展示模式（`display`）**：大屏宠物卡**禁用点击**，hover 时显示提示"教练已暂停操作"，无法进入个人页
- 个人页内部**不再需要**展示模式横幅（因为展示模式下根本无法进入个人页）

## FIFA 能力卡风格调整

现有 `FifaPlayerCard.vue` 是暗黑风格（`#1a1a2e` 背景 + 金色边框）。在个人页中需要调整为明亮风格：

- **背景**：白色 `#FFFFFF`，圆角 16px，阴影
- **边框色**：保留金银铜分级逻辑（≥85金 `#FFD700`、≥70银 `#C0C0C0`、其他铜 `#CD7F32`）
- **文字**：主文字深灰 `#333`，次要文字 `#666`
- **雷达图**：调整配色为明亮色系
- **进度条**：高绿 `#4caf50`、中黄 `#ff9800`、低红 `#f44336`

通过 props 传入 `theme: 'dark' | 'light'`，组件内部根据 theme 切换配色方案。

## 数据模型

### 复用现有 API
- `GET /player/:playerId/pet` — 宠物数据
- `GET /player/:playerId/mode` — 模式（用于个人页是否需要禁用操作）
- `GET /coach/player-stats/:playerId` — FIFA 能力卡数据（综合评分、雷达图、维度分数、排名）

### 大屏模式 API 补充
- `GET /public/mode/:phone` — 新增公开接口，大屏页面获取教练模式（无需认证）

## 视觉风格

### 色彩
- **页面背景**：浅蓝 `#E3F2FD` 到浅绿 `#E8F5E9` 渐变
- **卡片背景**：白色 `#FFFFFF`，圆角 16px，阴影 `0 4px 12px rgba(0,0,0,0.08)`
- **阶段边框**：egg灰、baby绿、teen黄、adult橙、rare粉
- **按钮**：喂食=橙色 `#ff9800`，训练=蓝色 `#2196f3`

### 字体
- 标题：ZCOOL KuaiLe，22px bold
- 评分数字：JetBrains Mono / Russo One，48px
- 正文：系统默认，14-16px

### 动画
- 宠物 emoji：浮动动画（3s ease-in-out infinite）
- 卡片入场：淡入上浮，stagger 100ms
- 积分变化：飘字动画

## 响应式适配

| 断点 | 布局 | 字体缩放 |
|---|---|---|
| ≥1024px（大屏/电视） | 左右分栏 55%/45% | 1.5x（确保远距离可读） |
| 768-1024px（平板） | 垂直堆叠，卡片宽度 80% 居中 | 1.2x |
| <768px（手机） | 垂直堆叠，全宽 | 1x |

## 大屏页面模式控制（补充修改）

`TeamScreenPage.vue` 需要补充以下逻辑：

1. 页面加载时调用 `GET /public/mode/:phone` 获取教练模式
2. 将模式状态传递给 `TeamPetCard` 组件
3. `TeamPetCard` 根据模式调整点击行为：
   - `open`：正常点击跳转个人页
   - `display`：点击无反应，显示 toast "教练已暂停操作"
4. 宠物卡 hover 时添加 `cursor: not-allowed`（display 模式）

```json
  "success": true,

}
```
